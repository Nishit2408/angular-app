# E-commerce Application Front-End

The application should be built using [Angular 9](https://angular.io/), [Bootstrap 4](https://getbootstrap.com/docs/4.4/getting-started/introduction/) and [ng-boostrap](https://ng-bootstrap.github.io/#/home). It is an Angular based front-end for the e-commerce Application called **luv2shop**. .
